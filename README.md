
# Random-Isolation Threat Detection (RITD)
A Hybrid Machine Learning Framework for Securing Virtual Machines Against Advanced Cyber Threats using:

- Adaptive Bitterling Fish Optimization (ABFO)
- Deep Belief Network (DBN)
- Variational Density Peak Clustering (VDPC)
- Federated Learning-based training architecture

This repository provides a full implementation for research reproducibility.
